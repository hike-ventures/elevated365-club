03 — Usage Guidelines

How to apply the brand

Clear rules for maintaining visual consistency across every platform and format where Elevate Dynamics 365 has a presence.

🎨

Primary colour

Use #263FC9 (Primary Blue) as the main colour for CTAs, links, and featured elements. It should never occupy more than 40% of total visual surface area.

🌑

Dark backgrounds

#182465 (Deep Navy) is reserved for headers, footers, hero sections, and backgrounds that require maximum contrast with white text.

🔤

Type hierarchy

Styrene A always leads the visual hierarchy (headings, nav, CTAs). Tiempos Text stays in body copy and messages. Never reverse the roles.

💬

Chat & messaging

Chat messages always use Tiempos Text at 16px with a line-height of 1.75. Timestamps and message metadata use Styrene A at caption size.

✓ Do

  • Maintain a minimum contrast ratio of 4.5:1 for text on coloured backgrounds
  • Use the defined type scale without in-between sizes
  • Apply Deep Navy (#182465) for primary text on white backgrounds
  • Respect white space: minimum 48px between sections
  • Use Styrene for navigation and Tiempos for content

✗ Don't

  • Modify the HEX values of the official palette
  • Mix Styrene and Tiempos within the same text block
  • Use the lighter blues as text colour on white backgrounds
  • Apply unauthorised gradients over brand colours
  • Use alternative typefaces outside the defined fallbacks

Buttons

All buttons use Styrene A Medium at 14px with a border-radius of 10px.
